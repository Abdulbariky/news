## How to run my website locally

### prerequisite 

- node.js installed
- git installed

### steps

1. clone repository;  
- git clone https://github.com/your-username/your-repository.git

2. Navigate to the directory;
- cd your-directory

3. install dependencies
- npm install

4. start the server
- npm start

this should automatically open your browser
